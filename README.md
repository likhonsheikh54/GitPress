# GitPress

GitPress is a modern, Git-based static site CMS that offers a WordPress-like experience with improved performance and security.

## Features

- Git-based content management
- Static site 
- WordPress-like admin interface
- SEO optimization
- Responsive design
- Dark mode support
- Custom theming
- Plugin architecture
